# 🏏 IPL Match Winner Predictor

## 📌 Overview
Predicts IPL match winners using 6 supervised ML algorithms trained on real match data (2008–2022).

## 🤖 Algorithms Compared
| Algorithm | Accuracy |
|---|---|
| Random Forest | 41.1% |
| Neural Network | 40.0% |
| Decision Tree | 36.3% |
| KNN | 34.7% |
| SVM | 31.6% |
| Logistic Regression | 17.9% |

## 🏆 Best Model: Random Forest (41.1%)

Python, Pandas, Scikit-learn, Matplotlib
